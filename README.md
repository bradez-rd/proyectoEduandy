# Proyecto Eduandy

Práctica del parcial de Programación Orientada a Objetos en C#.

## Temas incluidos

- Sintaxis básica y variables.
- Estructuras de control.
- Clases y objetos.
- Herencia.
- Polimorfismo.
- Encapsulamiento.
- Interfaces y abstracción.

## Ejecutar el proyecto

Se requiere .NET 8 o una versión posterior compatible.

```powershell
dotnet run
```

El programa solicitará un número entero y ejecutará los siete ejercicios en orden.

## Antes de entregar

1. Cambiar los nombres y edades de ejemplo en `Program.cs`.
2. Completar los nombres y matrículas en `INTEGRANTES.md`.
3. Ejecutar `dotnet run` y comprobar el resultado.
